##OBJECTIVES:


(i)   To reduce food wastage and support needy communities

(ii)  To provide a centralized platform for surplus food donations

(iii) To enable real-time food donation posting and tracking

(iv)  To provide location-based food pickup using Google Maps

(v)   To improve system security using OTP verification

(vi)  To ensure food safety through automated expiry removal

(vii) To analyze donation impact using statistical dashboards
